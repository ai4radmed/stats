# NM-Statistics 데이터 파이프라인 설계

이 문서는 다양한 소스(API, 엑셀 파일 등)에서 발생하는 데이터를 수집, 가공 및 통합하여 데이터베이스에 저장하는 구조를 정의합니다.

## 1. 데이터 소스 및 수집 방식

| 소스명 | 형태 | 수집 방식 | 빈도 |
| :--- | :--- | :--- | :--- |
| **공공데이터포털 API** | JSON/XML | 서버 사이드 API Fetch | 매월/분기별 자동 갱신 |
| **대한핵의학회(KSNM)** | Excel (.xlsx) | 관리자 페이지 내 수동 업로드 | 연 1~2회 (비정기) |
| **사용자 입력** | UI Form | 직접 입력 (필요 시) | 수시 |
