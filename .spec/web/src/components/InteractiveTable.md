# 공통 UI 컴포넌트: InteractiveTable 명세

## 1. 개요
모든 데이터 테이블에서 공통으로 사용되는 필터링 및 정렬 기능을 갖춘 대화형 테이블 컴포넌트의 표준을 정의합니다.

## 2. 주요 기능
- **컬럼별 정렬(Sorting)**: 
    - 헤더 클릭 시 오름차순/내림차순 정렬 전환.
    - 정렬 상태 시각적 표시 (화살표 아이콘 등).
- **컬럼별 필터링(Filtering)**:
    - 텍스트 입력 또는 셀렉트 박스를 통한 실시간 데이터 필터링.
    - 여러 컬럼의 복합 필터 지원.
- **200줄 원칙 준수**:
    - 테이블 헤더, 바디, 필터 컨트롤을 모듈화하여 각 파일의 복잡도를 낮춤.

## 3. 구현 가이드 (Vanilla JS + Astro)
- **로직 관리**: 필터링 및 정렬 로직은 클라이언트 사이드 스크립트 또는 URL 쿼리 파라미터를 통해 관리합니다.
- **성능**: 데이터가 1,000건 이하인 경우 클라이언트 사이드 필터링을 우선 적용합니다.

## 4. 시각적 요구사항
- **Premium Design**: 호버 효과, 부드러운 전환 애니메이션, 명확한 포커스 상태.
- **반응형**: 모바일 환경에서도 가로 스크롤 또는 카드형 전환을 통해 가독성 유지.
