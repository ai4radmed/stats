# 품목별 실적 현황 페이지 명세 (performance.astro)

## 1. 개요
방사성의약품의 연도별 생산 및 수입 실적을 시각화하고 상세 데이터를 필터링/정렬하여 조회할 수 있는 페이지입니다.

## 2. 데이터 요구사항
- **출처**: Supabase `mfds_item_performance` 테이블 (마스터 테이블과 Join)
- **필수 필드**: 연도(year), 제품명(item_name), 업체명(entp_name), 구분(result_part), 금액(amount).

## 3. 기능 명세
- **연도별 요약**: 특정 연도를 선택하여 해당 기간의 실적만 집중 조회.
- **실적 정렬**: 금액(amount) 기준 내림차순 정렬을 기본값으로 설정.
- **실적 구분 필터**: '생산'과 '수입' 실적을 분리해서 확인 가능.
- **200줄 원칙 준수**:
    - `PerformanceTable.astro`: 테이블 렌더링 로직 분리.
    - `PerformanceFilters.astro`: 필터 UI 로직 분리.
    - `performance.astro`: 전체 레이아웃 및 데이터 패칭 담당.

## 4. UI/UX 요구사항
- **금액 표시**: 천 단위 콤마(,) 표시 및 단위(원) 명시.
- **시각적 강조**: 실적이 높은 상위 항목에 대해 별도의 배지나 강조 효과 적용.
- **빈 데이터 처리**: 실적이 없는 경우 "해당 조건의 실적 데이터가 없습니다" 메시지 표시.
